# 🌱 UpcycleConnect

Plateforme de recyclage et d'upcycling connectée - Projet Annuel 2A5 2026

## 🚀 Stack Technique

- **Frontend**: React 18
- **Backend**: Go (API REST)
- **Base de données**: PostgreSQL 15
- **Reverse Proxy**: Nginx
- **Domaine**: https://upcycleconnect.net

## 📦 Installation
```bash
git clone https://github.com/Kaazhar/PA_2a5_2026.git
cd PA_2a5_2026
cp .env.example .env
docker compose up -d
```
